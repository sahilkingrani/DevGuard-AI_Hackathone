# Autonomous SRE Agent

A small Python demo that simulates a crashing server and uses **Google Gemini** to analyze crash logs and automatically patch `app.py`.

## Overview

| File | Role |
|------|------|
| `app.py` | Simulated server app with an intentional typo (`DB_ENPOINT` vs `DB_ENDPOINT`). On crash, writes a traceback to `crash_log.txt`. |
| `agent.py` | Autonomous SRE agent: reads the log and source, asks Gemini for a structured fix, then overwrites `app.py` with the corrected code. |
| `crash_log.txt` | Generated by `app.py` when it crashes (input for the agent). |
| `.env` | API key for Gemini (not committed). |

## Prerequisites

- Python 3.10+
- A [Google AI / Gemini API key](https://aistudio.google.com/apikey)
